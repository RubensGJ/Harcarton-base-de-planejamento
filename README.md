# Harcarton Base de Planejamento

Base de planejamento do projeto haCARthon, organizada como repositório de apoio para estudo, referência e avaliação de proposta.

## Conteúdo

- `haCARthon - *.pdf`: edital, guia e briefings oficiais
- `TranscriçãoVideo1 (1).txt`: transcrição usada como fonte complementar
- `links uteis.txt`: links de apoio levantados para a pesquisa
- `pesquisa-ferramentas-agentes-haCARthon.md`: pesquisa prévia de ferramentas e agentes
- `.agents/`: agentes e papéis de trabalho do projeto
- `relatorios/`: relatórios produzidos durante a análise do escopo

## Observações

Este repositório guarda materiais de planejamento e análise. Arquivos temporários, caches locais e resíduos de execução do Codex ficam fora do versionamento por `.gitignore`.
